# Arena Institucional - Site & Landing Pages

**Stack:** Vite + HTML5 + vanilla JavaScript + Tailwind CSS v4 + CSS customizado premium
**Agente:** Para qualquer trabalho visual, invocar `frontend-arena`

## Regras Rápidas

- Fontes: Bebas Neue (títulos), Barlow/Barlow Condensed (corpo/UI) via Google Fonts
- Paleta: Navy (#0a1d58), Navy Dark (#060f2e), Orange (#ffaa00), Off-white (#e8edf8)
- Animações: reveal (fade-up), glow pulse, orbit — sempre com IntersectionObserver
- Responsividade: Mobile First, botões min 48px, Lucide Icons
- Textura: grid overlays sutis, glow effects, nunca fundos chapados
- Specs completas: ler `.skills/skill_design_premium_arena.md` antes de codificar
